Privacy Policy

Last updated: 2026-04-21

Overview

Hermes Bot is a Facebook Page management tool operated for personal use. This policy describes how we handle data accessed through the Facebook API.

Data Collection

This application accesses Facebook Page data including posts and comments solely for the purpose of page management. No personal data is collected, stored, or shared with third parties.

Data Usage

Data accessed via the Facebook API is used only to perform the requested actions (publishing posts, reading comments, replying to comments) and is not retained beyond the scope of each request.
